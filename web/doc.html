Introducción

Coffe CSS, la mejor opción orientada al negocio cafetero para darle un estilo caracteristico a su pagina web.

Inicio rápido

Si necesitas agregar rapidamente coffe css a tu proyecto puedes utilizar jsDelivr que es un CND gratuito de código abierto. También existe una sección de descargas en donde encontrarás los archivos fuente.

Descarga

Descarga Cofee CSS para obtener el código compilado de CSS & JavaScript, o incluyelo con tus administradores de paquetes favoritos como npm, RubyGems y más.

Archivos fuente

Compile Coffe CSS con su propia canalización de activos descargando nuestros archivos fuente. Si necesita nuestro conjunto completo de herramientas de compilación, se incluyen para desarrollar Coffe CSS y sus documentos, pero es probable que no sean adecuadas para sus propios fines.

Colores del tema

Usamos un subconjunto de colores para crear una paleta de colores más pequeña para generar una mejor visualización de color. Entre nuestra gama de colores tenemos los colores Primarios, Secundarios y Ligeros. Coffe CSS esta respaldado por una extensa gama de colores que tematizan nuestros estilos y componentes.

Tipografía

Nuestra tipografia se basa en Coffe and tea, que es una tipografía que va acorde al estilo de una cafetería y se muestra agradable a la vista. La tipografía se establece con los rems, por lo que los tamaños de fuente y las relaciones espaciales se pueden ajustar según el tamaño de una sola propiedad de tamaño de fuente. Cabe mencionar que todas las medidas siguen siendo de base 10, por lo que un < h1 > con un tamaño de fuente de 5,0 rem solo significa 50 px.
