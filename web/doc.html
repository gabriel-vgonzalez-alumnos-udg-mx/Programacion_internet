Introducción

Coffe CSS, la mejor opción orientada al negocio cafetero para darle un estilo caracteristico a su pagina web.


Inicio rápido

Si necesitas agregar rapidamente coffe css a tu proyecto solo incluye los paquetes a tu proyecto y agrega la siguiente etiqueta en el head de tu archivo html.


<link rel="stylesheet" href="css/styles.css">


<script src="js/scripts.js"></script>

Colores

Usamos un subconjunto de colores para crear una paleta de colores más pequeña para generar una mejor visualización de color. Entre nuestra gama de colores tenemos los colores Primarios, Secundarios y Ligeros.


Tipografía

Nuestra tipografia se basa en Coffe and tea, que es una tipografía que va acorde al estilo de una cafetería y se muestra agradable a la vista. La tipografía se puede establecer con la clase coffeeFont o en los eestilos con font-family. Cabe mencionar que si hay algun error aplicando la tipografía Coffee And Tea se aplicara Poppins o en su defecto san-serif.

Las medidas de los titulos estan establecidas desde 25pt hasta 12pt reduciendo de 2pt aprox, aunque pueden ser modificados desde las hojas de estilo.

Animaciones


Cambiar tamaño

Esta animación permite cambiar el tamaño de un contenedor div al pasar el cursor.

HTML 
<div class="cuadrado"></div>
CSS 
.cuadrado {
    width: 100px;
    height: 100px;
    background: #9e6730;
    color: #fff;
    display: flex;
    justify-content: center;
    align-items: center;
    transition: 1s;
}
.cuadrado:hover{
    transform: scale(2);
}

Cambiar color

Esta animación permite el cambio constante de color de un div.

HTML 
<div class="Color"></div>
CSS 

.Color{
    width: 100px;
    height: 100px;
    background: var(--base);
    animation:  cambiaColor 5s infinite;
    display: flex;
    justify-content: center;
    align-items: center;
    color: #fff;
}
                       
@keyframes cambiaColor{
    10%{ background: #fff; }
    30%{ background: #f3dbc3; }
    50%{ background: #e8c39e; }
    60%{ background: #9e6730; }
    80%{ background: #804000; }
    90% { background: black;  }
}

Anime.js

Ofrecemos los paquetes de anime.js para que puedas utilizarlo en tus animaciones.


Ejemplo:


HTML 
<div class="anime" id="special"></div>
CSS 
.anime{
    width: 100px;
    height: 100px;
    background-color: var(--accentcolor);
}
JS 
var special = anime({
    targets: document.getElementById('special'),
    translateX: 200,
    autoplay: true
});
                          
document.querySelector('.anime').onclick = special.restart;
